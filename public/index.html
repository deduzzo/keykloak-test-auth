ASP Messina TEST SPID

Applicazione demo per collaudo AGID

Test Autenticazione SPID

Questa applicazione demo permette di verificare il flusso di autenticazione SPID tramite il Service Provider Keycloak di ASP Messina.
Premi il pulsante per autenticarti con SPID.

Entra con SPID Reset sessione e Logout da Keycloak

Cosa viene testato:

  • Flusso SAML 2.0 tra SP (Keycloak) e IdP SPID
  • Ricezione e visualizzazione attributi SPID
  • Conformita metadata SP alle regole tecniche AGID
  • Integrazione OIDC tra app e Keycloak
Dettagli tecnici
Architettura:
App (OIDC) → Keycloak (SAML SP) → IdP SPID

Entity ID: https://login.asp.messina.it/realms/asp
SPID Metadata: /realms/asp/spid/metadata
Client OIDC: spid-test-login (public)
Protocollo App: OpenID Connect - Authorization Code Flow
Piattaforma: Keycloak 26.x + spid-provider plugin
